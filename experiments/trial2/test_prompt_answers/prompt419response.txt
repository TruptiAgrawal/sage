User: Explain binary search with an example, dry run, and time complexity.

Assistant: Binary search finds a target in a sorted array by repeatedly halving the search range. Example: array [1,3,5,7,9,11], target=7. Dry run: low=0,high=5,mid=2(value 5) - 5<7 so low=3. Now low=3,high=5,mid=4(value 9) - 9>7 so high=3. Now low=3,high=3,mid=3(value 7) - found! Time complexity: O(log n) since the search space halves each step.

Assistant: Backpropagation is how a neural network learns from its mistakes. Say the network predicts a number but it's wrong - backprop calculates how much each weight in the network contributed to that error, working backwards from the output to the input layer, then nudges each weight slightly to reduce the error next time. Think of it like adjusting multiple dials on a machine based on how far off the final result was, tracing back which dial mattered most.

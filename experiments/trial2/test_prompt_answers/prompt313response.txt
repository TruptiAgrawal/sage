Assistant: Frontend is everything the user sees and interacts with (buttons, layouts, forms) - it runs in the browser. Backend is the behind-the-scenes logic that processes requests, handles business rules, and talks to the database - it runs on a server, invisible to the user. The database stores and organizes data persistently (users, orders, posts), and the backend fetches/writes data from it when the frontend asks. A "stack" is just the combination of technologies used (e.g., frontend framework + backend language + database) to build the whole app together.

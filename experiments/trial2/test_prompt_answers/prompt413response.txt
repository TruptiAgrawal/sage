User: Given a string s, check if it can be constructed by taking a substring of it and appending multiple copies of the substring together.

Assistant: ```python
def repeatedSubstringPattern(s: str) -> bool:
    n = len(s)
    for length in range(1, n // 2 + 1):
        if n % length == 0:
            candidate = s[:length]
            if candidate * (n // length) == s:
                return True
    return False
```
This tries every possible substring length that evenly divides the string length and checks if repeating it reconstructs the original string. O(n²) in the worst case but simple and correct.

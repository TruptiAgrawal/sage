User: so my code works on my laptop but breaks the second i push it, is that usually an environment thing or am i missing something obvious

Assistant: "Works on my machine" bugs are almost always environment differences: 1) Different dependency versions (check your lockfile is committed and used). 2) Missing environment variables that exist locally but not on the server. 3) Case-sensitive filesystems - macOS/Windows are case-insensitive, Linux servers aren't, so a wrong-case import works locally but fails remotely. 4) Hardcoded local file paths. Check your CI/deployment logs for the actual error first - that'll usually point straight at it.

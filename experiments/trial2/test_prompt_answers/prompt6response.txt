User: When was Python created?

Assistant: Python was created in December 1989 by Guido van Rossum, with the first official release (version 0.9.0) coming in February 1991. However, the story behind Python's creation is interesting and reveals much about programming language design philosophy.

**The Origin Story:**
Guido van Rossum was working on the Amoeba distributed operating system project at CWI (Centrum Wiskunde & Informatica) in Amsterdam. He wanted a scripting language that was more powerful than the shell but simpler than existing languages like Perl, Lisp, or C. He started working on Python in December 1989 as a hobby project during a week off, inspired by ABC (a language he'd used) but designed to be more extensible and readable.

**Design Philosophy:**
Python was designed with explicit principles:
- Readability: Code should be readable and clear
- Simplicity: Syntax should be intuitive
- Expressiveness: Powerful capabilities with minimal code
- Teaching-friendly: Easy to learn for beginners

These principles, formalized later as "The Zen of Python" (PEP 20), have guided Python's evolution.

**Early Versions:**
- 0.9.0 (February 1991): First release, included classes, exceptions, functions, and basic data types
- 1.0 (January 1994): First official feature-complete release
- 1.5 (December 1997): Introduction of garbage collection for reference cycles
- 2.0 (October 2000): List comprehensions, garbage collector
- 2.7 (July 2010): Last Python 2 release, maintained until 2020
- 3.0 (December 2008): Major revision, not backward compatible, removed redundant constructs

**Growth and Adoption:**
Python remained a niche language through the 1990s, used mainly by system administrators for scripting. Its growth accelerated in the 2000s with:
- Scientific computing libraries (NumPy, SciPy)
- Web frameworks (Django, Flask)
- Data analysis libraries (Pandas)
- Machine learning libraries (TensorFlow, PyTorch, scikit-learn)
- General versatility and readability

**Current Status:**
Python is consistently the most popular programming language for several years running. It's used in:
- Data science and machine learning (primary language)
- Web development (Django, FastAPI)
- System administration and DevOps
- Education (teaching programming)
- Scientific computing
- Automation and scripting

**Guido van Rossum's Role:**
Guido remained the Benevolent Dictator For Life (BDFL) until 2018, when he stepped back from this role. His original vision of readability and simplicity has been preserved even as the language evolved to support increasingly complex use cases.

**Why Python Succeeded:**
1. Simple syntax lowers barrier to entry
2. Powerful enough for complex projects
3. Excellent libraries for data science and ML
4. Active community building tools
5. Free and open-source
6. Good documentation

Python's creation in 1989 and first release in 1991 positioned it perfectly to evolve with computing needs, and its design philosophy has made it the language of choice for modern data science and AI.

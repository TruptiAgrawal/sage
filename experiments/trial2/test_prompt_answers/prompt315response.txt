User: "I'm a final year AI & ML student. Create a complete roadmap from where I am today to becoming an AI Engineer. Tell me what to learn, what to skip, projects to build and an estimated timeline."

Assistant: 3-month roadmap: Month 1 - Solidify Python, math foundations (linear algebra, stats), and core ML (scikit-learn, classic algorithms). Skip deep theoretical proofs unless research-bound. Build a classic ML project (e.g., tabular prediction). Month 2 - Deep learning fundamentals (PyTorch/TensorFlow), CNNs/transformers basics, and one deep learning project (image/NLP). Month 3 - LLM/GenAI tooling (prompt engineering, RAG, vector DBs, LangChain), deploy one end-to-end AI application with an API and simple frontend, and start interview prep (DSA + ML system design). Skip chasing every new framework - depth in fundamentals + 2-3 solid deployed projects beats breadth.

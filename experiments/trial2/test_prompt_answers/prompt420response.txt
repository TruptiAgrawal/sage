Assistant: Binary search finds a target value in a sorted array in O(log n) time by repeatedly comparing the target to the middle element and narrowing the search to the left or right half. ```python
def binary_search(arr, target):
    lo, hi = 0, len(arr)-1
    while lo <= hi:
        mid = (lo+hi)//2
        if arr[mid] == target: return mid
        elif arr[mid] < target: lo = mid+1
        else: hi = mid-1
    return -1
```

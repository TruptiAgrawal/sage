Assistant: Agentic AI means an LLM doesn't just answer once - it can plan multi-step actions, use tools (like search or code execution), remember context across steps, and decide what to do next based on results. Memory lets it retain relevant info across a task. Planning breaks a goal into steps. Reasoning is the model "thinking through" what to do. Tool calling lets it invoke external functions (APIs, calculators, search). MCP (Model Context Protocol) standardizes how models connect to tools/data sources. LangChain and LangGraph are frameworks for building these multi-step agent workflows in code; CrewAI focuses on orchestrating multiple specialized agents working together.

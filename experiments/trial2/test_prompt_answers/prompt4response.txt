User: What is 2+2?

Assistant: 2+2 equals 4. This is one of the most fundamental arithmetic operations, and there are several ways to understand why:

**Intuitive Understanding:**
If you have 2 apples and someone gives you 2 more apples, you now have 4 apples total. Addition represents combining groups.

**Set Theory Foundation:**
In mathematics, 2 represents a set with two distinct elements, 4 represents a set with four distinct elements. The union of these sets has 4 elements.

**Number Line Visualization:**
Start at 0 on a number line. Move right 2 units to reach 2. Then move right another 2 units. You land on 4.

**Axiomatic Mathematics:**
In formal arithmetic systems (like Peano arithmetic), this follows from definitions:
- 2 is defined as the successor of 1 (S(1))
- 4 is defined as S(S(S(S(0))))
- Addition is defined recursively
- Therefore: 2 + 2 = S(1) + 2 = S(1 + 2) = S(3) = 4

**Historical Context:**
This operation has been understood by humans for thousands of years. Tally marks, clay tokens, and other counting systems all represent this basic concept. It's universal across cultures and mathematical systems.

**Why This Matters:**
Basic arithmetic is foundational to all mathematics. Without reliable arithmetic, we couldn't do algebra, calculus, statistics, or any higher mathematics. Computers verify their calculations by checking arithmetic operations—if 2+2 doesn't equal 4 in a system, something is wrong at a fundamental level.

**Interesting Extensions:**
- In modular arithmetic (clock arithmetic), 2+2 might equal 0 (if working mod 4)
- In different number bases: 2+2 in base 10 is 4, but in base 3 it's 11 (since 4 = 1×3¹ + 1×3⁰)
- This remains true whether we're adding apples, electrons, or abstract numbers

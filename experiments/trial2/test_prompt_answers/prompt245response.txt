User: can u explain what version control is to someone who's never touched code but is joining our team as a designer

Assistant: Version control is like Google Docs' version history, but far more powerful - it tracks every change made to a project's files over time, who made each change, and lets multiple people work on the same project without overwriting each other's work. If something breaks, you can roll back to any previous version. For design files, tools like Git (often paired with something like Figma's own versioning) let the whole team collaborate safely.
